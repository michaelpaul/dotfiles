Michael Paul's dotfiles
=====================

Meus arquivos de configuração.

## Utilização

Instalação.

```
git clone https://github.com/michaelpaul/dotfiles.git
cd dotfiles
./install
```

Atualização.

```
git pull
./install
```

Desinstalar (util em mudanças de caminhos).

```
./uninstall
```
